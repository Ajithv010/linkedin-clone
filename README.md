# How to Download and Extract

1. Click on the ZIP file in the repository (or click **Code → Download ZIP**).
2. After download, right-click on the ZIP file.
3. Select **Extract All…** and unzip it.
4. Open the extracted folder.

5. ## How to Run the Backend


# Open terminal / command prompt
cd backend

# Install dependencies
npm install   # or yarn install

# Start backend server
npm run dev    




# Open a new terminal window
cd frontend

# Install dependencies
npm install   # or yarn install

# Start the frontend
npm run dev     

# Technologies Used

# Frontend
- React.js / HTML / CSS / JavaScript
- Axios / Fetch – For API calls
- Bootstrap / Tailwind CSS

# Backend**
- Node.js + Express.js and Django 
- MongoDB / MySQL / Firebase
- REST API Architecture
- JSON-based Communication



# Features of the Project

 **User Authentication** (Login / Signup) 
 **Data stored and fetched from database**  
 **Fully functional backend API**  
 **Interactive and responsive frontend UI**  
 **CRUD Operations (Create, Read, Update, Delete)** 
 **Frontend & Backend connected via API**  
 **Error handling & validation implemented**  
 **Clean folder structure (frontend + backend separated)**  



# Notes

 Run the backend first, then the frontend
